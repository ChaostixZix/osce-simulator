# CLI OSCE Prototype (Archived)

This folder contains the deprecated Node.js CLI prototype and earlier PHP experiment.
The active application lives in `webapp/` (Laravel + Vue via Inertia).

Contents
- JavaScript CLI: `app.js`, `lib/`, `utils/`, `cases/`, tests under `test/`.
- Legacy PHP prototype: `app/`, `config/medical_training.php`, `database/migrations/`.
- Bootstrap scripts: `start.sh`, `start.bat`.
- Environment samples: `.env.example`, `.env.medical_training.example`.

Notes
- Do not commit secrets. Any `.env` here is for local development only.
- The CLI is no longer maintained. Refer to `webapp/README.md` for current setup.
